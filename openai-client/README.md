

# openai-client

Java libraries for using OpenAI's GPT APIS.

## Usage

```xml
<dependency>
    <groupId>io.github.hamawhitegg</groupId>
    <artifactId>openai-client</artifactId>
    <version>0.1.12</version>
</dependency>
```
* Java 17 or later
